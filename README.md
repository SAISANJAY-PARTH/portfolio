# Sai Sanjay - Premium Financial Analytics Portfolio

This is a premium, lightweight, responsive portfolio website built with pure HTML, custom CSS, and vanilla JavaScript. 

## 📂 Project Structure
- `index.html`: Contains the layout sections (Hero, About, Skills, Experience, Projects, Certifications, Contact) and a hidden, print-optimized resume view.
- `style.css`: Contains custom styling themes (dark/light toggles), grid systems, animations, and `@media print` rules.
- `script.js`: Drives interactive elements (typewriting intro, theme storage, project filtering, mobile drawer, scroll highlights).

## 🚀 Key Features
1. **Dynamic Volatility & Return Clustering Hero Animation**: A visual card in the hero mockup representing your real-world ML market regime model.
2. **Interactive Filtering**: Visitors can filter your projects between "FinTech & ML", "BI & Analytics", and "Web & Education".
3. **Flawless Resume Print / PDF Export**: 
   - A dedicated print icon triggers `window.print()`.
   - The CSS hides navigation bars, color themes, contact forms, and renders **only** a clean, single-page, black-and-white CV designed exactly like your professional resume.
   - Anyone visiting your website can click "Print Resume" and select "Save as PDF" to download your resume locally on the fly!
4. **Local Theme Storage**: Saves users' dark/light preferences in the browser.

## 🛠️ How to Customize or Host
- **To Customize Text/Links**: Open `index.html` and search for the respective sections.
- **To Customize Styling**: Modify root colors at the top of `style.css`.
- **To View Locally**: Double-click `index.html` to open it in any web browser.
- **To Host**: Upload these files directly to GitHub Pages, Netlify, Vercel, or any host of your choice!
